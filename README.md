## Overview

The Storyteller Chatbot is an interactive platform that brings stories to life. Users can **type a prompt**, and the assistant will respond with a **narrative story**. Each story is enhanced with **immersive visuals** and **spoken audio**, allowing users to **read, watch, and listen** to the story in a seamless experience.

---

## Features

- **Interactive Storytelling:** Users engage in a conversation with a friendly assistant that only tells stories.  
- **Audio Narration:** Stories can be played aloud directly within the interface for an immersive experience.  
- **Illustrated Scenes:** Each story comes with a calming, visually rich illustration to enhance the narrative.  
- **User-Friendly Interface:** Designed to be intuitive, with chat, audio, and image display all in one place.  
- **Relaxing Experience:** Focused on creating a cozy, calm, and engaging storytelling environment.
